# Google Assistant Service

Google Assistant voice commerce

## Features

- ✅ Full API integration with Google Assistant
- ✅ Order synchronization
- ✅ Inventory management
- ✅ Webhook handling
- ✅ Real-time updates
- ✅ Production-ready

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

```bash
export GOOGLE_ASSISTANT_API_KEY="your_api_key"
export GOOGLE_ASSISTANT_API_SECRET="your_api_secret"
export PORT=8104
```

## Running

```bash
python main.py
```

## API Documentation

Visit `http://localhost:8104/docs` for interactive API documentation.
