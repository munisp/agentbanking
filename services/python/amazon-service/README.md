# Amazon Service

Amazon Marketplace integration

## Features

- ✅ Full API integration with Amazon
- ✅ Order synchronization
- ✅ Inventory management
- ✅ Webhook handling
- ✅ Real-time updates
- ✅ Production-ready

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

```bash
export AMAZON_API_KEY="your_api_key"
export AMAZON_API_SECRET="your_api_secret"
export PORT=8098
```

## Running

```bash
python main.py
```

## API Documentation

Visit `http://localhost:8098/docs` for interactive API documentation.
