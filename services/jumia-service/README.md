# Jumia Service

Jumia Africa marketplace integration

## Features

- ✅ Full API integration with Jumia
- ✅ Order synchronization
- ✅ Inventory management
- ✅ Webhook handling
- ✅ Real-time updates
- ✅ Production-ready

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

```bash
export JUMIA_API_KEY="your_api_key"
export JUMIA_API_SECRET="your_api_secret"
export PORT=8101
```

## Running

```bash
python main.py
```

## API Documentation

Visit `http://localhost:8101/docs` for interactive API documentation.
