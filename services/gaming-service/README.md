# Gaming Service

Gaming platforms (Discord/Steam) commerce

## Features

- ✅ Full API integration with Gaming
- ✅ Order synchronization
- ✅ Inventory management
- ✅ Webhook handling
- ✅ Real-time updates
- ✅ Production-ready

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

```bash
export GAMING_API_KEY="your_api_key"
export GAMING_API_SECRET="your_api_secret"
export PORT=8100
```

## Running

```bash
python main.py
```

## API Documentation

Visit `http://localhost:8100/docs` for interactive API documentation.
