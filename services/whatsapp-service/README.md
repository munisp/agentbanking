# Whatsapp Service

WhatsApp Business API integration

## Features

- ✅ Full API integration with Whatsapp
- ✅ Order synchronization
- ✅ Inventory management
- ✅ Webhook handling
- ✅ Real-time updates
- ✅ Production-ready

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

```bash
export WHATSAPP_API_KEY="your_api_key"
export WHATSAPP_API_SECRET="your_api_secret"
export PORT=8080
```

## Running

```bash
python main.py
```

## API Documentation

Visit `http://localhost:8080/docs` for interactive API documentation.
